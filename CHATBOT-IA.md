## 🎨 **Design et animations**

### **Bouton flottant**
- **Position** : Coin inférieur droit
- **Couleurs** : Dégradé orange → rouge NSY
- **Badge IA** : Cerveau rotatif pulsant
- **Particules** : 3 points lumineux flottants
- **Tooltip** : "Assistant IA NSY"

### **Interface de chat**
- **Dimensions** : 384px × 500px (responsive)
- **Header** : Dégradé NSY avec statut en ligne
- **Messages** : Bulles différenciées user/bot
- **Avatar bot** : Icône robot avec indicateur vert
- **Saisie** : Zone de texte + bouton envoi animé

### **Animations**
- **Apparition** : Spring animation du bouton (délai 2s)
- **Ouverture** : Scale + slide up de la fenêtre
- **Messages** : Fade in + slide up pour chaque message
- **Typing** : 3 points pulsants réalistes
- **Hover effects** : Micro-interactions sur tous les éléments

## 🔧 **Intégration technique**

### **Composant principal**
```jsx
<ChatBot />
```

### **Position dans l'app**
```jsx
// Dans App.jsx
<Footer />
<ScrollToTop />
<ChatBot />  // ← Ajouté ici
```

### **État du composant**
- `isOpen` : Contrôle l'ouverture du chat
- `messages` : Historique des conversations
- `inputValue` : Texte en cours de saisie
- `isTyping` : Simulation de réponse IA

### **Fonctions clés**
- `getBotResponse()` : Analyse et génère les réponses
- `handleSendMessage()` : Traitement des messages utilisateur
- `scrollToBottom()` : Auto-scroll des nouveaux messages
